# 🔧 SETUP GUIDE (VERY IMPORTANT)

## 1️⃣ Get API Key

Go to:
https://console.groq.com

👉 Sign up  
👉 Create API key  

---

## 2️⃣ Add Environment Variable in Vercel

Go to your project → Settings → Environment Variables

Add:

GROQ_API_KEY = your_key_here

---

## 3️⃣ Deploy

Push your project to GitHub  
Import into Vercel  
Deploy
